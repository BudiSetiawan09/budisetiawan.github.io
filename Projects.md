## M365 Tenant Migration and Azure AD Integration

**Client:** Multi-site business | **Timeline:** June 2025

### Challenge
Migrate from on-premise to cloud-first infrastructure with zero downtime for 15+ users across multiple locations.

### Solution
- Deployed M365 tenant from scratch
- Configured SharePoint environment with document libraries
- Set up Azure AD (Entra ID) and joined all devices
- Migrated MX records for email routing
- Configured Outlook profiles for 15+ users
- Synchronized on-premise file shares to SharePoint

### Technologies Used
`M365` `Azure AD` `SharePoint Online` `DNS Management` `Outlook` `OneDrive`

### Outcome
Zero-downtime migration with all users operational within 48 hours. Full cloud integration achieved with hybrid identity management.

## Enterprise CCTV Infrastructure Deployment

**Client:** Resort property | **Timeline:** January 2026

### Challenge
Deploy 13-camera surveillance system integrated with existing network infrastructure for 24/7 security monitoring.

### Solution
- Deployed 13 PoE cameras across property
- Configured Uniview NVR with RAID storage
- Integrated with core network switch using VLAN segmentation
- Set up remote viewing capability for management
- Configured retention policies and motion detection
- Implemented PoE+ switch for camera power delivery
- Created dedicated VLAN for surveillance traffic isolation
- Applied QoS policies to prioritize recording traffic

### Technologies Used
`Uniview NVR` `PoE Networking` `VLAN Configuration` `Network Segmentation` `QoS` `RAID Storage`

### Outcome
24/7 surveillance system with 30-day retention, remote access capability, and network-isolated security infrastructure.

## Critical Network Recovery - Firewall HA Implementation

**Client:** Financial services firm | **Timeline:** August 2025

### Challenge
Restore internet service after critical outage and implement redundancy to prevent future failures for business-critical operations.

### Solution
- Diagnosed primary WAN failure through systematic troubleshooting
- Configured Sophos firewall high-availability pair
- Set up automatic failover between primary/secondary units
- Validated failover scenarios under simulated failure conditions
- Migrated WAN connections for full redundancy

### Technologies Used
`Sophos Firewall` `HA Clustering` `WAN Failover` `BGP Routing` `Network Redundancy`

### Outcome
Sub-5-minute service restoration with automatic failover now preventing single points of failure. Business continuity ensured for critical financial operations.
